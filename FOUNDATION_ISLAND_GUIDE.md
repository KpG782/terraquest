### üèÜ Challenges

#### Challenge 1: Install Terraform
**Difficulty**: Beginner  
**Points**: 50  
**Estimated Time**: 15-20 minutes

**Objective:**
Install Terraform on your local machine and verify the installation is working correctly.

**What You'll Do:**
1. Download Terraform from official website
2. Install on your operating system (Windows/macOS/Linux)
3. Add to PATH environment variable
4. Verify installation with `terraform version`
5. Test basic commands

**Prerequisites:**
- A computer with admin/sudo access
- Internet connection

**Validation:**
- Successfully run `terraform version`
- See version information displayed
- All basic commands work

**Navigation:**
- Click "Start Challenge" from module page
- Follow step-by-step instructions
- Use hints if you get stuck
- Mark as complete to earn 50 points

---

#### Challenge 2: Your First Configuration
**Difficulty**: Beginner  
**Points**: 100  
**Estimated Time**: 30-40 minutes

**Objective:**
Write and apply your first Terraform configuration file to create a local file resource.

**What You'll Do:**
1. Create a project directory
2. Write a `main.tf` configuration file
3. Initialize Terraform (`terraform init`)
4. Plan the changes (`terraform plan`)
5. Apply the configuration (`terraform apply`)
6. Explore the state file
7. Make changes and update
8. Clean up with `terraform destroy`

**Prerequisites:**
- Terraform installed (Challenge 1 completed)
- Text editor
- Basic command line knowledge

**Validation:**
- Successfully initialize Terraform
- Create a local file resource
- View and understand the state file
- Update the resource
- Destroy resources cleanly

**Navigation:**
- Accessible after Challenge 1
- Complete step-by-step guide provided
- Hints available if needed
- Mark as complete to earn 100 points
