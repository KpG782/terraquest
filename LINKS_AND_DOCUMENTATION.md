## üéÆ Navigation & Controls

### Keyboard Shortcuts
```
Tab              ‚Üí Navigate to next module
Shift + Tab      ‚Üí Navigate to previous module
Enter / Space    ‚Üí Select focused module
Escape           ‚Üí Return to map view
```

### Mouse Controls
```
Left-click + drag    ‚Üí Rotate camera view
Right-click + drag   ‚Üí Pan camera view
Scroll wheel         ‚Üí Zoom in/out
Click on region      ‚Üí Navigate to module
Hover over region    ‚Üí View module details
```

### Touch Controls (Mobile/Tablet)
```
One-finger drag      ‚Üí Rotate view
Two-finger pinch     ‚Üí Zoom in/out
Two-finger drag      ‚Üí Pan view
Tap on region        ‚Üí Select module
```

### Camera Boundaries

## üìä Progress Tracking

### Automatic Saving
- Progress saved to browser's localStorage
- Persists across sessions
- Includes:
  - Completed modules
  - Current module
  - Unlocked achievements
  - Overall progress percentage

### Achievement System
Unlock achievements by:
- Completing your first module
- Finishing entire regions
- Completing all challenges
- Reaching the Workflow Summit

### Points System
Earn points through challenges:
- **Beginner**: 50-100 points
- **Intermediate**: 125-175 points
- **Advanced**: 200-400 points
- **Total Available**: 2,000+ points
